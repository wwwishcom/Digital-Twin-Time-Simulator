# Digital Twin Time Simulator

> 현실 시간 사용 패턴을 복제한 디지털 트윈 기반 미래 일정 시뮬레이션 AI

## Overview

**Digital Twin Time Simulator (Twin Time)** 는
사용자의 목표, 행동, 루틴 데이터를 기반으로
현재 상태를 모델링하고 미래 결과를 시뮬레이션하는
Digital Twin 기반 자기관리 플랫폼입니다.

단순 일정 관리 앱이 아니라,

“지금의 행동이 목표 날짜에 어떤 결과를 만들 것인가?”

를 수치로 보여주는 시스템입니다.

## Core Concept

Twin Time은 사용자의 삶을 두 개의 차원으로 모델링합니다.

### 1️. Goal Twin (성과 차원)

1. 프로젝트 기반 목표 관리

2. 세부 할 일(Task) 단위 추적

3. 예상 소요 시간 기반 진행률 계산

4. 현재 페이스 기준 목표일 도달 가능성 시뮬레이션

5. 일정 기간 무진도 시 Momentum Drop 감지

**지금 속도로 목표에 도달할 수 있는가?** 에 초점을 맞췄으며 

타 서비스들과 달리 그룹 프로젝트 기능이 활성화되어있습니다.

### 2. Self-care Twin (상태 차원)

1. 수면 기록

2. 운동 기록

3. 감정 기록

4. 루틴 연속성(스트릭)

5. 친구와 비교 기능 (선택 공개)

6. Twin Wallet (재정 관리)

성과와 분리된 독립 영역으로
개인의 균형과 지속 가능성을 관리합니다.
